Facebook OSINT & Network Penetration Testing

X

October 2023

This project demonstrates external intelligence gathering and internal network testing. Part 1 covers Facebook OSINT, and Part 2 covers VM network enumeration and penetration testing.

Part 1 — Facebook OSINT & DNS Reconnaissance

  • Tools: Whois, Nslookup, Dnsrecon, TheHarvester, Dnsenum, Dig, Dmitry, BuiltWith, Shodan, Recon-ng
  • Identified registrar, admin contacts, authoritative name servers, MX, CNAME, SOA records
  • Discovered subdomains (developers.facebook.com, business.facebook.com, etc.)
  • Attempted DNS zone transfer (success via zonetransfer.me)
  • Analyzed web technologies and server stack
  • Generated OSINT HTML report using Recon-ng
❮ ❯

Part 2 — VM Network & Penetration Testing

  • Tools: Kali Linux, Metasploitable2, Nmap, Wireshark, macchanger
  • Identified VM IPs, MAC addresses, and subnet mask (/24)
  • Spoofed MAC addresses using macchanger
  • Captured network traffic with Wireshark
  • Performed Nmap host discovery, ACK/FIN scans, and full -A enumeration
  • Ran NSE scripts (safe & vuln) to identify potential vulnerabilities
❮ ❯

Outcome

Demonstrates ability to conduct both external reconnaissance via OSINT and internal network scanning in a lab environment, showing real-world ethical hacking techniques.
